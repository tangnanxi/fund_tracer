# Fund Tracker Skill

查询订阅基金的实时估算净值与涨跌幅。

## 数据来源

- 天天基金网（East Money）公开接口：`https://fundgz.1234567.com.cn/js/{code}.js`
- 返回单只基金的实时估算净值（`gsz`）、估算涨跌幅（`gszzl`）、估值时间（`gztime`）等。
- 仅在中国大陆交易时段内有实时估算数据；非交易时段可能返回上一交易日收盘净值。

## 使用方法

### 1. 配置基金代码

编辑同目录下的 `funds.txt`，把你订阅的基金代码写进去，每行一个：

```text
008114
166002
002009
```

也可以带备注（脚本只读取每行第一个空白符前的内容）：

```text
001632 天弘食品饮料
005827 易方达蓝筹
```

### 2. 运行查询

#### 方式 A：读取配置文件

```bash
cd ~/.kimi-code/skills/fund-tracker
python fund_tracker.py
```

#### 方式 B：临时传入代码

```bash
python fund_tracker.py 001632 005827 003095
```

### 3. 在 Kimi 中调用

直接对我说：

> “查一下我的基金”

我会读取 `~/.kimi-code/skills/fund-tracker/funds.txt` 并返回实时涨跌。

## 文件说明

| 文件 | 说明 |
| --- | --- |
| `SKILL.md` | 本说明文档 |
| `fund_tracker.py` | 查询脚本 |
| `funds.txt` | 你的基金代码配置 |

## 注意事项

- 接口返回的是**估算净值**，可能与当晚发布的实际净值有偏差。
- 若基金已停牌或不在交易时段，`gszzl` 可能为 `0` 或空。
- 网络不稳定时可能请求失败，可稍后重试。
